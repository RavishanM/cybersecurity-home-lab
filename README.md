# Beginner Cybersecurity Home Lab

## Project Overview

This project is a beginner cybersecurity home lab using Kali Linux in Oracle VirtualBox, a Mac, and a Windows PC. The goal was to practice basic networking and security concepts in a safe environment using only my own devices.

## Lab Setup

- Kali Linux VM running in Oracle VirtualBox
- Mac used as a target device
- Windows PC used as another target device
- All devices connected to the same home network
- Kali network mode set to Bridged Adapter

## What I Practiced

- Finding the Kali VM IP address
- Testing network connectivity with ping
- Scanning devices with Nmap
- Finding open ports on a Mac
- Checking listening services directly on the Mac
- Checking Windows port 445
- Understanding open, closed, and filtered ports
